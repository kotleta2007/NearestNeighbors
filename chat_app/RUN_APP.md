# Steps to run app
- cd to your_path/NearestNeighbors/chat_app
- pip install -r requirements.txt
- open terminal in current directory and write: 
    - for macos/linux:
        - chmod +x start_app.sh
        - ./start_app

    - for windows:
        - start_app
    
    - incase any error:
        - streamlit run main.py
